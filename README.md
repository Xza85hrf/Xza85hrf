## 👋 Hey, I'm Ravi

I'm an IT Support Specialist who also develops automation tooling.
Over time, simple helper scripts became a suite of utilities — system
monitors, AI-powered transcription, a UPS watchdog on a Raspberry Pi,
and an image-generation pipeline.

My focus is converting repetitive, time-consuming tasks into reliable,
hands-off solutions that improve uptime and cut manual effort.

## 🧠 How I think about tooling

> **Automate the pain points, not the occasional task.** One-off
> scripts are fine; anything repeated belongs in reliable automation.

> **Ship honest docs.** Every claim in my README is backed by existing
> code; future ideas stay in the CHANGELOG's *Unreleased* section, not
> the feature list.

> **Discipline before the commit.** Tests run, GPU support is verified
> on real hardware, and each commit is one logical change.
